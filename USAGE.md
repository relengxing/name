# 使用说明

## 新功能介绍

### 1. 🎯 生成按钮置顶
生成按钮现在放在控制面板的最上方，方便快速操作。

### 2. ✍️ 可编辑文本框 + 智能分隔符识别
在"生成结果"区域添加了一个可编辑的文本框，你可以：
- **粘贴外部名字列表**：从其他地方复制名字，直接粘贴到文本框中
- **智能识别分隔符**：自动识别多种分隔符（逗号、顿号、空格、换行等）
- **手动编辑**：直接在文本框中添加、删除或修改名字
- **实时更新**：文本框内容变化后会自动更新卡片展示（有0.5秒延迟）

#### 智能分隔符规则：

**中文模式**（中文字符占比>30%）：
- 支持：换行、空格、逗号(`,`/`，`)、顿号(`、`)、分号(`;`/`；`)、制表符、竖线、斜杠
- 示例：`张三,李四、王五 刘六` → 解析为4个名字

**英文模式**（中文字符占比≤30%）：
- 支持：换行、逗号(`,`/`，`)、分号(`;`/`；`)、制表符、竖线、斜杠
- **不分割空格**（保留英文名中的空格）
- 示例：`John Smith, Jane Doe` → 解析为2个完整名字

### 3. 🔀 多级排序组合
现在支持最多 3 种排序方式的组合：

**排序逻辑**：
- 先按第一排序规则排序
- 如果第一排序相同，再按第二排序规则
- 如果第二排序也相同，再按第三排序规则

**示例场景**：
1. **长度 + 字母顺序**
   - 第一排序：长度顺序（短→长）
   - 第二排序：字母顺序（A-Z）
   - 结果：先按名字长度排序，相同长度的按字母顺序排列

2. **笔画 + 长度 + 字母**
   - 第一排序：笔画顺序（少→多）
   - 第二排序：长度顺序（短→长）
   - 第三排序：字母顺序（A-Z）
   - 结果：先按笔画数排序，笔画相同的按长度排，长度也相同的按字母排

### 4. 📋 新增顿号分隔符
导出分隔符新增"顿号（、）"选项，常用于中文名字的分隔。

**所有分隔符选项**：
- 换行
- 逗号 (,)
- **顿号 (、)** ← 新增
- 空格
- 分号 (;)
- 制表符 (Tab)

## 完整使用流程

### 方式一：生成名字
1. 在控制面板选择生成模式（实时生成/名字库随机）
2. 选择语言类型（中文名/英文名）
3. 设置生成数量（1-100）
4. 点击"🚀 开始生成"按钮
5. 名字会自动显示在文本框和卡片区域

### 方式二：粘贴名字（支持智能识别）
1. 从Excel、网页或其他地方复制名字列表
2. 直接粘贴到"生成结果"区域的文本框中
3. 系统会**自动识别分隔符**并解析
4. 支持多种分隔符混合使用

**示例**：
```
# 中文（支持多种分隔符混合）
张三,李四、王五 刘六；赵七|钱八

# 英文（保留名字中的空格）
John Smith, Jane Doe; Bob Lee | Alice Wang
```

### 排序操作
1. 在文本框下方选择排序方式（可选择1-3个排序规则）
2. 点击"✨ 应用排序"按钮
3. 名字会按照设定的规则重新排序
4. 排序后的结果会同步更新到文本框和卡片展示

### 复制导出
1. 选择导出分隔符（换行、逗号、顿号等）
2. 点击"📋 复制全部"按钮
3. 所有名字会按选定的分隔符格式复制到剪贴板
4. 或者直接点击单个名字卡片复制该名字

## 高级技巧

### 1. 混合使用生成和手动输入
- 先点击生成按钮创建一批名字
- 在文本框中手动添加或删除部分名字
- 应用排序重新整理

### 2. 从不同来源导入
- **Excel**：选中一列，复制，粘贴（自动识别换行）
- **网页**：复制名字列表，粘贴（自动识别各种分隔符）
- **逗号分隔文本**：直接粘贴，自动解析

### 3. 快速测试不同排序
- 输入或生成名字后
- 依次尝试不同的排序组合
- 查看哪种排序最符合需求

### 4. 智能分隔符的妙用
```
# 一次性粘贴多种格式混合的名字
张三,李四、王五 刘六
赵七；钱八|孙九/周十

# 系统自动识别所有分隔符，解析为10个独立的名字
```

## 注意事项

1. **智能识别**：系统会自动判断是中文还是英文，应用不同的分隔规则
2. **英文名空格**：纯英文模式下，空格不会作为分隔符（保留"John Smith"这样的完整名字）
3. **中英混合**：如果中文字符较多，会按中文规则分割（包括空格）
4. **实时更新**：手动编辑文本框后，会有0.5秒的延迟才更新卡片显示
5. **排序组合**：排序选项可以设置为"无"，这样该级排序不会生效
6. **去重处理**：生成的名字会自动去重，但手动输入的名字不会自动去重

## 常见问题

**Q: 如何清空所有名字？**
A: 直接清空文本框的内容即可。

**Q: 为什么修改文本框后卡片没有立即更新？**
A: 为了性能考虑，设置了0.5秒的防抖延迟。你也可以点击"应用排序"按钮立即更新。

**Q: 多级排序如何工作？**
A: 先按第一排序规则排序，遇到相同的情况时再用第二排序规则，以此类推。

**Q: 为什么我的英文名字被分成了两部分？**
A: 如果文本中有较多中文字符（>30%），会触发中文模式，空格会作为分隔符。建议分开处理纯中文和纯英文的列表。

**Q: 支持哪些分隔符？**
A: 
- 中文模式：换行、空格、逗号、顿号、分号、制表符、竖线、斜杠
- 英文模式：换行、逗号、分号、制表符、竖线、斜杠（不包括空格）

**Q: 可以混合使用多种分隔符吗？**
A: 可以！系统会自动识别所有支持的分隔符。例如：`张三,李四、王五 刘六`

**Q: 如何导出特定格式？**
A: 在"导出分隔符"中选择需要的格式（换行/逗号/顿号/空格等），然后点击"复制全部"。
