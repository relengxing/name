# 名字生成器 ✨

这是一个精美的名字生成器网页应用，具有现代化的玻璃拟态设计风格。集成 **Faker.js** 库，支持15+种语言/地区的真实名字生成。

## 🚀 快速部署

**本项目已完全配置好 GitHub Pages 部署！**

👉 [点击查看部署指南](START_HERE.md) 或 [快速部署](QUICK_DEPLOY.md)

只需 3 步即可部署到 GitHub Pages：
1. 创建 GitHub 仓库
2. 推送代码
3. 启用 GitHub Pages

详细步骤请查看 [START_HERE.md](START_HERE.md)

## 功能特点

- 🎨 **现代化设计**: 使用 Tailwind CSS 和 Glassmorphism 玻璃拟态效果，蓝绿渐变色主题
- 🌍 **多语言支持**: 支持15+种语言/地区（中文、英文、日文、韩文、法文、德文等）
- 🚀 **Faker.js 集成**: 使用专业的 Faker.js 库生成真实、多样化的名字
- 📚 **大型名字库**: 内置近7000个中文名字和多个英文名字，避免重复
- 🖌️ **精准笔画计算**: 集成 cnchar.js 库，准确计算中文笔画数
- 📊 **灵活排序**: 支持多种排序方式（最多3级排序组合）
  - 字母顺序（A-Z / Z-A）
  - 笔画顺序（少→多 / 多→少）
  - 长度顺序（短→长 / 长→短）
- 🔢 **可调数量**: 自定义生成名字的数量（1-100）
- ✍️ **可编辑文本框**: 支持从外部粘贴名字列表
- 🧠 **智能分隔符识别**: 自动识别多种分隔符（逗号、顿号、空格、换行等）
- 📋 **分隔符选项**: 支持换行、逗号、顿号、空格、分号、制表符等多种分隔符
- 📋 **便捷复制**: 一键复制单个或全部生成的名字
- 🔍 **SEO 优化**: 完整的 SEO 元标签配置

## 使用方法

### 基本使用
1. 在控制面板选择语言/地区
2. 设置生成数量（1-100）
3. 点击"🚀 开始生成"按钮
4. 系统会从名字库中随机选择（中文）或使用 Faker.js 生成（其他语言）

### 排序功能
1. 可以选择最多3级排序方式组合
2. 点击"✨ 应用排序"按钮
3. 支持字母、笔画、长度等多种排序

### 粘贴外部名字
1. 从Excel、网页等地方复制名字列表
2. 直接粘贴到文本框
3. 系统自动识别分隔符并解析
4. 可以对粘贴的名字进行排序

## 名字库

名字库文件存放在 `names/` 文件夹中：

- `chinese_names.txt` - 中文名字库（每行一个名字）
- `english_names.txt` - 英文名字库（每行一个名字）

你可以在这些文件中添加更多名字，每行一个，保存后刷新页面即可使用。

## 部署到 GitHub Pages

本项目已配置 GitHub Actions 自动部署。

### 步骤：

1. 将代码推送到 GitHub 仓库
2. 进入仓库的 Settings → Pages
3. 在 "Source" 下选择 "GitHub Actions"
4. 推送代码到 `main` 分支，GitHub Actions 将自动构建和部署

### 配置统计（可选）：

部署后建议配置网站统计：
- 📊 [配置百度统计和 Google Analytics](ANALYTICS_SETUP.md)
- 替换 `index.html` 中的统计 ID
- 获取网站访问数据和用户分析

### 手动部署：

也可以在 GitHub 仓库的 Actions 标签页手动触发部署。

详细步骤请查看 [DEPLOYMENT.md](DEPLOYMENT.md)

## 特色功能

### 🎯 智能分隔符识别
自动识别输入文本的语言类型，应用相应的分隔规则：
- **中文模式**：识别逗号、顿号、空格、换行等

支持最多3种排序方式的组合：
- 先按第一排序规则排序
- 相同时按第二排序规则
- 再相同时按第三排序规则

- 数万个真实名字数据
- 15+种语言/地区支持
- 自动适配文化命名习惯
- 支持中间名、前缀、后缀等变体
